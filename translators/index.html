TwitchBot for every language

Our goal is to create a community of Twitch users who don't have to worry about the language barrier. We want to translate TwitchBot's UI into multiple other languages, and we need the community's help to do so.

How it works

We'll send you a collection of Python files through GitHub that have the existing sentences in English. All you have to do is translate the English parts to the languge that you've applied for.

class Errors:
 err_report = "Please report this error to the developers at https://link.twitchbot.io/support."
 forbidden = emoji.cmd_fail + "I don't have the correct permissions to do that."

For example, if you're translating to Japanese, you'll translate the file to something like this:

class Errors:
 err_report = "https://link.twitchbot.io/support で開発者にこのエラーを報告してください"
 forbidden = emoji.cmd_fail + "そのための正しい権限がありません。"

Translator benefits

All translators will receive a lifetime subscription to TwitchBot Premium as a thank you for helping us make TwitchBot available to your language.
